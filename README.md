MEKAN is an AI-powered industrial assistant designed to help technicians
diagnose and repair machines quickly using image capture, audio input,
and technical data analysis. Features  📸 Capture machine codes via camera  🎙️ Voice-based issue reporting  🤖 AI-powered diagnostics  🧩 Spare parts identification  🔒 Data privacy for factoriesHow It Works Capture --- Scan code, record audio, or type the issue\ Analyze --- AI matches data with engineering database\Tech Stack  Flutter (Mobile App)  FastAPI (Backend)  OpenAI API (AI Engine)  Firebase (Auth & Storage)Business Model  SaaS subscriptions for factories  Commission from spare parts suppliers via MEKAN QR Enterprise licensing
And also other income from other sources later on.
